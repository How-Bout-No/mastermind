# Mastermind Heist Editor
Simple Heist Editor for [YimLuaAPI](https://github.com/TupoyeMenu/YimLuaAPI), compatible with both Legacy and Enhanced. Crafted from my contributions to [Samurai's Scripts](https://github.com/YimMenu-Lua/Samurais-Scripts).

<br>

### Install
Place `Mastermind_vX.lua` into `%APPDATA%/YimLuaAPI/scripts/`
